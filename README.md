# 🎓 Student Depression Data Analysis Dashboard

An interactive **Tableau Dashboard** developed to analyze the key factors contributing to **student depression** using demographic, academic, financial, and lifestyle data. The project leverages **SQL Server (T-SQL)** for data preprocessing and **Tableau** for building interactive visualizations that help identify trends and patterns affecting student mental health.

---

## 📌 Project Overview

Student mental health has become a growing concern across educational institutions. This project analyzes multiple factors such as **Academic Pressure, Financial Stress, Study Hours, Study Satisfaction, Sleep Duration, Gender, Age Group, Dietary Habits, and Family History** to uncover meaningful insights.

The dashboard enables users to interactively explore the relationships between these variables and understand the factors associated with student depression.

## ✨ Key Insights

- Academic pressure has a noticeable relationship with student mental health.
- Financial stress contributes significantly to depression among students.
- Students with lower study satisfaction show a higher likelihood of depression.
- Sleep duration influences overall mental well-being.
- Study hours vary considerably across different student groups.
